# web audio microphone streaming to raspberry pi speakerz


setup
```
sudo apt-get install node npm
sudo npm install -g bower
npm install
bower install
```

run socket server
```
node server.js  
```
-or-
```
npm start
```

run web server
```
python -m SimpleHTTPServer 8000
```
-or-
```
sudo python -m SimpleHTTPServer 80
```
