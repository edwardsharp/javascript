# dmx websocket ctrl

useful thing for controlling a usb-dmx device via web app.

setup
```
sudo apt-get install node npm
npm install
bower install
```

run socket server
```
node server.js  
```
-or-
```
npm start
```

run web server
```
python -m SimpleHTTPServer 8000
```
-or-
```
sudo python -m SimpleHTTPServer 80
```
