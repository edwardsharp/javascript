DHTMLX Gantt and Angular Demo
===================

Demo of Gantt chart component for Angular framework

### How to start


 - clone the repository or download files
 - install dependencies
~~~ 
npm install 
~~~

 - run server
~~~
ng serve
~~~

 - open http://localhost:4200 in browser
